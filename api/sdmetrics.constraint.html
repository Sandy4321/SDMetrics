sdmetrics.constraint package¶

This module implements constraint checking which makes sure the statistical properties of the synthetic data match the specified metadata.

Classes:

Goal

This enumerates the goal for a metric; the value of a metric can be ignored, minimized, or maximized.

Metric(name, value[, tags, goal, unit, …])

This represents a single instance of a Metric.

Functions:

metrics(metadata, real_tables, synthetic_tables)

This function takes in (1) a sdv.Metadata object which describes a set of relational tables, (2) a set of “real” tables corresponding to the metadata, and (3) a set of “synthetic” tables corresponding to the metadata.
sdmetrics.constraint.metrics(metadata, real_tables, synthetic_tables)[source]¶

This function takes in (1) a sdv.Metadata object which describes a set of relational tables, (2) a set of “real” tables corresponding to the metadata, and (3) a set of “synthetic” tables corresponding to the metadata. It yields a sequence of Metric objects.

Parameters

  • metadata (sdv.Metadata) – The Metadata object from SDV.

  • real_tables (dict) – A dictionary mapping table names to dataframes.

  • synthetic_tables (dict) – A dictionary mapping table names to dataframes.

Yields

Metric – The next metric.
